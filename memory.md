# 안개 전쟁 (Fog of War) - 게임 설계 메모

## 게임 개요
- 보드게임 스타일의 실시간 1대1 온라인 대전 게임
- 턴제 전략 + 안개 전쟁 (서로의 위치를 모름, 전함 게임처럼)
- 기술 스택: Node.js + Express + Socket.io + Vanilla JS (프로토타입)
- 최종 목표: React + TypeScript + Canvas 픽셀아트 + PostgreSQL + Docker (CLAUDE.md 참고)

## 보드
- 5x5 정사각형 (25칸)
- 공유 보드이지만 서로 상대 말의 위치를 볼 수 없음
- 좌표: (col, row) 0-indexed, 표시는 1-indexed

## 플레이어
- 각 플레이어는 3개의 말을 사용 (1티어, 2티어, 3티어에서 각 1개 선택)
- 게임 시작 전 비공개로 말 3개 선정 (드래프트)
- HP 10개를 3개 말에 자유 분배 (최소 1, 최대 8)
- 게임 시작 후 캐릭터 스탯은 서로 공개 (위치만 비공개)

## 게임 흐름
1. 로비: 방 생성/입장 또는 AI 연습 모드
2. 드래프트: 각 티어에서 1개씩 비공개 선택
3. HP 분배: 총 10 HP를 3개 말에 분배
4. 배치: 5x5 보드에 자유 배치 (초기 배치 시 겹치기 불가)
5. 게임: 턴제 교대, 매 턴 이동 OR 공격 중 택 1
6. 승리: 상대방 말 전부 격파

## 이동 규칙
- 모든 말: 상하좌우(십자) 1칸만 이동 가능
- 대각선 이동 불가, 제자리 이동 불가
- 게임 중 말 겹침 가능 (초기 배치만 불가)
- 이동 시 상대에게는 "상대방이 이동했습니다"만 표시 (어떤 말이 어디로인지 비공개)

## 공격 규칙
- 공격 시 게임이 히트/미스 판정 → 결과 알려줌
- 자기 자신은 자기 공격에 피해를 받지 않음 (왕 등)
- 파괴된 적 말은 타입이 공개됨

## 캐릭터 목록

### 1티어 - 전체공격타입 (ATK 1)
| 캐릭터 | 아이콘 | 공격 범위 |
|--------|--------|-----------|
| 졸병   | ⚔     | 자신이 있는 칸 기준 대각선(/) 전체 공격 |
| 궁수   | 🏹    | 자신이 있는 칸 기준 가로줄 전체 공격 |
| 전차병 | 🪖    | 자신이 있는 칸 기준 세로줄 전체 공격 |

### 2티어 - 일부공격타입 (ATK 2)
| 캐릭터 | 아이콘 | 공격 범위 |
|--------|--------|-----------|
| 장군   | 🎖    | 인접 십자(+) 4칸 공격 |
| 암살자 | 🗡    | 자신의 칸 + 임의의 칸 1개 동시 공격 |
| 기사   | 🐴    | X자 대각선 인접 4칸 공격 |

### 3티어 - 상위공격력타입
| 캐릭터 | 아이콘 | 공격력 | 공격 범위 |
|--------|--------|--------|-----------|
| 왕자   | 👑    | ATK 3  | 좌우 인접 가로칸 공격 |
| 공주   | 🌸    | ATK 3  | 위아래 인접 세로칸 공격 |
| 왕     | ♛     | ATK 4  | 자신의 칸 공격 |

## 안개 전쟁 메커니즘
- 서로의 필드(말 위치)를 볼 수 없음
- 공격 결과(히트/미스)를 통해 상대 위치를 추리
- 이동 정보는 "이동했습니다" 정도만 공유 (어떤 말이 어디로인지는 비공개)

## AI 연습 모드
- 1인 플레이 가능
- AI가 자동으로 드래프트/HP분배/배치 수행
- AI는 추리 기반으로 플레이어를 추적 (개선 진행중)

## 방 관리
- 게임이 끝난 방(ended)은 자동 삭제 → 같은 이름으로 재생성 가능
- 대기 중 플레이어 나가면 방 삭제
