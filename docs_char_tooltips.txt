====================================
캐릭터 설명 팝업 (미니/풀) 전체 문구
====================================

[1티어]

#1 궁수 🏹
  스킬 [정비] SP 1 자유시전·1회
  설명: 공격 범위의 대각선 방향을 영구적으로 좌우 반전시킵니다. 이 스킬은 행동을 소비하지 않지만, 한 턴에 1회만 사용 가능합니다.

#2 창병 🔱
  스킬 없음
  설명: 세로줄 전체를 관통하는 공격 범위를 가집니다.

#3 기마병 🐎
  스킬 없음
  설명: 가로줄 전체를 관통하는 공격 범위를 가집니다.

#4 파수꾼 👁
  스킬 없음
  설명: 자기 주변 8칸 전부를 공격합니다. 공격력 0.5로 위력은 낮지만, 빈틈 없이 사방을 감시해 접근하는 적을 견제합니다.

#5 쌍둥이 강도 👬
  스킬 [분신] SP 2 행동소비형
  설명: 형과 동생 두 유닛이 한 세트로, 각각 독립적으로 이동·공격합니다. 스킬 사용 시 한쪽을 다른 쪽의 칸으로 합류시켜 업어옵니다. 이 스킬을 사용한 턴에는 다른 행동을 할 수 없습니다.

#6 척후병 🔭
  스킬 [정찰] SP 2 자유시전형
  설명: 상대방 말 1개의 행 또는 열 좌표를 랜덤으로 공개합니다. 이 스킬은 행동을 소비하지 않으며, SP가 있는 한 여러 번 사용 가능합니다.

#7 인간 사냥꾼 🪤
  스킬 [덫 설치] SP 2 행동소비형
  설명: 현재 위치에 몰래 덫을 설치합니다. 적이 해당 칸을 밟으면 덫이 발동하여 1 피해를 입힙니다. 이 스킬을 사용한 턴에는 다른 행동을 할 수 없습니다.

#8 전령 📯
  스킬 [질주] SP 1 자유시전·1회
  설명: 이동을 2회 실행할 수 있습니다. 이 스킬은 행동을 소비하지 않지만, 한 턴에 1회만 사용 가능합니다. 공격력 0.5로 전투력은 낮지만, 빠른 기동력으로 위치 선점이나 도주에 활용됩니다.

#9 화약상 💣
  스킬1 [시한폭탄 설치] SP 2 자유시전형
  설명: 주변 8칸 중 한 곳에 폭탄을 설치합니다. 이 스킬은 행동을 소비하지 않으며, SP가 있는 한 여러 개 설치가 가능합니다.
  스킬2 [기폭] SP 0 자유시전·1회
  설명: 설치한 폭탄을 모두 동시에 폭발시켜 해당 칸의 모든 적에게 1 피해를 줍니다. 이 스킬은 행동을 소비하지 않지만, 한 턴에 1회만 사용 가능합니다. 화약상이 사망하면 자동으로 발동됩니다.

#10 약초전문가 🌿
  스킬 [약초학] SP 3 자유시전형
  설명: 자신 주변 3×3 범위 내 아군의 체력을 +1 회복합니다 (자기 자신 제외). 이 스킬은 행동을 소비하지 않으며, SP가 있는 한 여러 번 사용 가능합니다.


[2티어]

#11 장군 🎖
  스킬 없음
  설명: 자기 자신을 포함한 십자 5칸을 공격합니다.

#12 기사 🗡
  스킬 없음
  설명: 자기 자신을 포함한 X대각선 5칸을 공격합니다.

#13 그림자 암살자 🗡
  스킬 [그림자 숨기] SP 1 자유시전·1회
  설명: 스킬 사용 시 다음 턴까지 공격과 상태 이상에 완전 면역이 됩니다. 이 스킬은 행동을 소비하지 않지만, 한 턴에 1회만 사용 가능합니다.

#14 마법사 🧙
  패시브 [인스턴트 매직]
  설명: 마법사는 피격 당할 때마다 1회용 SP를 제공합니다. 이 SP는 사용하면 사라집니다. 맞을수록 아군의 스킬 사용 기회가 늘어납니다.

#15 갑주무사 🛡
  패시브 [아이언 스킨]
  설명: 받는 피해가 항상 0.5 씩 감소해 생존력이 뛰어납니다. 다만 상태 이상으로 받는 피해는 감소되지 않습니다.

#16 마녀 🧙
  스킬 [저주] SP 3 행동소비형
  설명: 스킬 사용 시 체력이 1 이상인 적 한 명을 선택해 저주를 부여합니다. 이 스킬을 사용한 턴에는 다른 행동을 할 수 없습니다. 저주 상태의 적은 적의 차례마다 0.5씩 피해 받으며 스킬을 사용할 수 없습니다. 저주는 마녀가 죽거나, 저주 상태의 캐릭터 체력이 1 이하가 되면 해제됩니다.

#17 양손 검객 ⚔
  스킬 [쌍검무] SP 2 자유시전·1회
  설명: 스킬 사용 시 이번 턴 2회 공격할 수 있습니다. 이 스킬은 행동을 소비하지 않지만, 한 턴에 1회만 사용 가능합니다.

#18 쥐 장수 🐀
  스킬 [역병의 자손들] SP 2 자유시전형
  설명: 스킬 사용 시 보드 위 랜덤 3곳에 쥐를 소환합니다 (이미 쥐가 있는 곳에는 중복 소환 불가). 쥐가 있는 칸은 쥐 장수의 공격 범위에 포함됩니다. 이 스킬은 행동을 소비하지 않으며, SP가 있는 한 여러 번 사용 가능합니다.

#19 무기상 ⚒
  스킬 [정비] SP 1 자유시전·1회
  설명: 스킬 사용 시 가로에서 세로로, 혹은 세로에서 가로로 영구적으로 공격 방향을 전환합니다. 이 스킬은 행동을 소비하지 않지만, 한 턴에 1회만 사용 가능합니다.

#20 호위 무사 🛡️
  패시브 [충성] SP 2
  설명: 자신의 다른 왕실 태그 아군이 받을 공격 피해를 1로 줄이고, 그 피해를 자신이 대신 받습니다. 왕실 유닛의 방패 역할입니다. 왕실 유닛이 받게 될 상태 이상 또한 호위 무사가 대신 받습니다.


[3티어]

#21 왕자 👑
  스킬 없음
  설명: 자기 자신을 포함한 좌우 3칸을 공격합니다. 범위가 좁으므로 정확한 위치 운용이 생존의 핵심입니다.

#22 공주 👸
  스킬 없음
  설명: 자기 자신을 포함한 상하 3칸을 공격합니다. 범위가 좁으므로 정확한 위치 운용이 생존의 핵심입니다.

#23 국왕 ♛
  스킬 [절대복종 반지] SP 3 자유시전형
  설명: 스킬 사용 시 적 말 하나의 이름을 선언하고 위치를 지정하면, 해당 적을 강제로 이동시킵니다. 이 스킬은 행동을 소비하지 않으며, SP가 있는 한 여러 번 사용 가능합니다.

#24 드래곤 조련사 🐉
  스킬 [드래곤 소환] SP 5 자유시전·1회
  설명: 스킬 사용 시 HP 3, 공격력 3, 십자 5칸 범위공격을 가진 드래곤 유닛을 소환합니다. (보드 위 최대 1마리까지 소환 가능) 이 스킬은 행동을 소비하지 않지만, 한 턴에 1회만 사용 가능합니다.

#25 수도승 🙏
  스킬 [신성] SP 4 자유시전형
  설명: 아군 1명을 선택해 체력을 +2 회복하고 상태 이상을 제거합니다. 이 스킬은 행동을 소비하지 않으며, SP가 있는 한 여러 번 사용 가능합니다.
  패시브 [가호]
  설명: 악인 태그 적을 공격할 때 피해가 3으로 증가하고, 악인에게 피격 시 피해가 0.5로 감소합니다.

#26 학살 영웅 🪓
  패시브 [배반자]
  설명: 공격 시 범위 내 아군에게도 1 피해를 입힙니다. 아군 배치에 각별히 주의해야 합니다. 리스크 대비 리턴이 극단적인 양날의 검입니다.

#27 지휘관 📋
  패시브 [사기증진]
  설명: 좌우 각 1칸을 공격합니다. 인접한 아군의 공격력을 +1 버프합니다. 전선 뒤에서 아군을 강화하는 서포터 역할에 최적화되어 있습니다.

#28 유황이 끓는 솥 🔥
  스킬 [유황범람] SP 3 행동소비형
  설명: 스킬 사용 시 현재 보드의 테두리 전체에 피해 2의 대규모 공격을 가합니다. 이 스킬을 사용한 턴에는 다른 행동을 할 수 없습니다. 테두리에 몰린 적들을 한꺼번에 쓸어버리는 맵 병기입니다.

#29 고문 기술자 ⛓
  패시브 [표식]
  설명: 공격이 적중하면 대상에 표식을 부여합니다. 표식 상태의 적은 위치가 항상 공유됩니다.
  스킬 [악몽] SP 2 자유시전형
  설명: 표식이 찍힌 모든 적에게 피해 1을 줍니다. 이 스킬은 행동을 소비하지 않으며, SP가 있는 한 여러 번 사용 가능합니다. 여러 적을 표식으로 마킹한 뒤 일괄 타격하는 지속 딜러입니다.

#30 백작 🦇
  패시브 [폭정]
  설명: 상대 1~2티어 유닛에게 받는 피해가 0.5 감소합니다. 상대의 저티어 견제에 강한 내성을 가집니다. 후반으로 갈수록 빛나는 내구형 딜러입니다.
