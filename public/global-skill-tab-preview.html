글로벌 스킬 탭 — v2

사용자 이미지 피드백 반영판. 헤더 부가 설명 / SP 요약 제거. 스킬 없는 캐릭터는 카드에서 빠짐 (이론상 6명까지). 카드 컴팩트 + 상단 가능/불가 태그 + 박스 없는 컬러 유형 텍스트 + 행동소비형 카드도 초록 통일.

v1 → v2 변경 사항

  • ❌ 헤더 우측 SP 요약 / "직접 시전은 캐릭터 클릭으로" 부가 텍스트 제거 — 스킬 현황 + X 버튼 만
  • ❌ 액티브 스킬 없는 캐릭터 카드 노출 X (이론상 1v1 최대 6장)
  • ❌ 카드 내부 캐릭터 부가 설명 (티어 / HP) 제거 — 아이콘 + 이름만, 아이콘 크기 축소 (이름 폰트와 비슷하게)
  • ✅ 가능 / 불가 태그를 스킬 박스 상단에 부착 (교체됨·선택됨 스타일)
  • ✅ 스킬명 / 유형 / SP — 한 줄 인라인 컴팩트
  • ✅ 유형 텍스트 — 박스 제거, 색상 + 작은 도트 (●) 만
  • ✅ 게임 통일 용어: 자유시전형 / 자유시전·1회 / 행동소비형
  • ✅ 행동소비형 (마녀 저주) 도 붉은 테마 → 초록 테마 통일 (카드 / 박스 / 가능 태그)
  • ✅ 불가 사유 — 박스 내 중앙 정렬
  • ✅ 세로 패딩 / 마진 축소 — 공간 낭비 최소화
