🐀 쥐 소환 셀 — 3초 서서히 페이드

0~600ms 풀 강도 후 ~3초에 걸쳐 부드럽게 페이드.

셀 보라 → 투명 페이드 (3s) + orb 비행 + 본인🐀 (초록, left:7) / 적🐁 (빨강, left:2) 분기 확인

🌟 지휘관 사기증진 존 — 청록 + 버프 글로우 (상시)

지휘관 본인 셀 가장 진한 청록. 인접 셀은 지휘관 측이 가장 밝다가 외곽으로 갈수록 사라짐. 버프 받는 ally piece 가 셀에 있을 때만 청록 글로우 + ✦✧✨ 랜덤 파티클 (위치/타이밍/크기 모두 랜덤).

상시 박동 + 빛이 위로 솟아오름. 사운드: 패시브 톤.

🌿 약초학 시전 — 시계방향 + 벚꽃 + 꽃향기 마법 사운드

바로 위 셀부터 시계방향으로 빠르게 펼쳐짐. ✨🌸⭐ 벚꽃 + 반짝이가 위로 솟아오름. 글로우는 살짝 약해짐. 사운드: 펜타토닉 chime + 글리산도 + 향기 반짝이 + 마무리 종 (꽃향기 풍기는 귀여운 마법).

시계방향 reveal — 보드 가장자리에 잘리는 셀은 invisible.

🙏 수도승 신성 — 천천히 내려오는 빛 + 다수 반짝이

2.6s 천천히 내려오는 노란 빛 기둥 + ✝ 십자 떠오름 + 12개 ✨⭐✦ 별이 시간차로 흩날림.

3가지 성가대 합창 버전 — 듣고 선택해주세요.

🫅🏼 절대복종 반지 — 시전자/피해자 두 시점

게임 내부 ring-anim.js 를 그대로 호출 — 미리보기 전용 재구현 X. 셀·추리 토큰·설치물 (덫/폭탄/쥐) 모두 게임과 동일하게 처리. 모든 시나리오 끝나면 토큰/설치물은 그대로 유지됨.

⟶ 시전자 시점 (Caster)

적 piece 는 fog-of-war 로 안 보이지만 시전 시 자동 추리 토큰 이 도착 셀에 부착됨. 셀에 아군·설치물이 있어도 모두 정지 유지 — victim ghost 만 z-index:20 위에서 ring-arrived 후 ring-fade-out. 셀 페이드 는 alpha 기반 (자식 마커/토큰/설치물 미영향).

⟶ 피해자 시점 (Victim)

본인 piece 가 서버 commit 후 도착 셀에 이미 렌더됨. intro+vanish 동안 visibility:hidden (fromCell ghost 와 중복 방지), ring-arrived 시점에 자연스럽게 안착 (페이드 X).
