🎠 캐러셀 / 유해 스택 — 강제 이동(대표 페이지) 프리뷰

한 칸에 유닛·표식적·유해가 여럿이면 캐러셀. 각 슬롯은 일반 셀과 동일 크기·위치(유해 74px), HP 숫자 숨김.
새 액션 검출 시 규칙 페이지로 슬라이드 이동(애니메이션 O) → 회귀 없이 그 자리 고정. 우선순위: 내유닛(고티어)→동료(고티어)→적(고티어)→유해. 새 이동 진입(반지 포함)도 액션으로 카운트.
캐러셀 셀 (◀▶ 수동 넘김 가능)
비교용: 일반 셀 유닛 / 유해 (캐러셀 슬롯이 이 크기·위치와 동일)

구성

액션 (강제 이동 데모)

먼저 ◀▶ 로 다른 페이지로 넘겨 둔 뒤 아래를 눌러보세요. 보던 페이지가 달라도 액션 페이지로 슬라이드 이동 → 그 자리 고정(회귀 없음).

같은 칸: 내 유닛 공격 + 표식 적 피격

한 칸에 내 장군 + 표식 마녀(적)가 공존. 내 장군이 (다른 곳을) 공격하는 순간, 같은 칸의 표식 적이 피격당하는 상황. 세 가지 해석을 비교해보세요.

이동 규칙

  • 액션 검출(피격·공격·사망·새 이동진입) → 해당 페이지로 슬라이드 이동(애니 O).
  • 여럿 동시 → 대표 1명만: 내유닛→동료→적→유해, 각 그룹은 고티어 우선.
  • 이동 후 회귀 없이 그 자리 고정 — 다음 액션이 오면 다시 그쪽으로 슬라이드.
  • 수동 ◀▶ 로 본 페이지도 액션 전까지 그대로 유지.
  • 안 보고 있어도 액션은 실시간 진행 — 도착하면 진행 중 모션 그대로 보임.
