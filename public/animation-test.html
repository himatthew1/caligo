A. 프로필 카드 효과
profile-hit · 피격
유닛이 공격받았을 때 — 빨간 흔들림 + 금색 플래시 (1.8s)
heal-flash · 회복

profile-protected · 보호됨
공격이 들어왔지만 0 피해(호위무사 가로채기 / 아이언스킨 / 폭정 등). 기본은 v3 (1회 sweep + 둔둔 펄스).
curse-active · 저주 상시
저주 상태 캐릭터 — 진한 보라 배경 + 화염 불씨 파티클 상승
B. 데미지 도장 (3종)
dmg-stamp · 빨강 (직접 피해)
일반 피격 — 카드 우상단 빨간 사선 도장. 턴 시작까지 유지.
dmg-stamp.curse · 보라 (저주 지속)
저주 상태에서 매턴 0.5씩 누적. 턴 동안 유지.
dmg-stamp.loyalty · 파랑 (충성 대신 받음)
호위무사가 왕실 아군 대신 받은 피해. 작은 '충성' 태그 부착.
⭐ 본체 + 충성 동시 — 빨강 + 파랑
본체도 직접 맞고 동시에 동료 대신 받은 케이스. 두 도장 위치 분리됨.
hp-bar-damage · HP 바 빨간 오버레이
이번 턴 받은 피해를 HP 바 위 빨간 슬라이스로 표시.

상단 빨간 배너 펄스 (1.2s).
E. 스킬 시전 연출
⭐ detonation · 기폭 (3단계)
설치된 폭탄이 ① 서서히 노출 → ② 빨간 박동(arming) → ③ 폭발 플래시. 총 ~2.5s. 메시지·피해는 애니메이션 종료 후.
⭐ sp-spend-attn · SP 차감 집중 (섹션 글로우, 2s)
스킬 시전 시 SP 섹션이 부드럽게 박동(파랑→보라). 메인 연출은 마법구 비행이며, 섹션 글로우는 보조 강조용.
⭐ sp-orb-transfer · 마법구 비행 + 숫자 tick
출발 시 시전자 -1 / 도착 시 상대 +1. 페어리 트레일 더스트 + tick 효과음. 인스턴트는 위로 흩날림.

스킬 시전 시 인스턴트 SP가 먼저 사라지고 0.5초 텀 후 일반 SP 마법구 비행 시작.
⭐ instant-gain · 인스턴트 매직 (마법사 SP 합류)
마법사 인스턴트 매직 발동 시 황금 마법구가 SP 숫자로 합류 → 도착 흡수.

shrink-intro · 보드 파괴 카운트다운

마법 구슬 두 개가 좌우 대칭 페어리 동선으로 입장 → 텍스트 프레임 형성 → 각 SP 숫자로 분리 비행.
V1 사이드: 좌우 측면에서 부드러운 곡선 입장 (현재 기본)
V2 코너: 모서리에서 나선형으로 모임

V4 호버: 화면 아래에서 부드럽게 떠오르며 입장

I. 신규 (Day 2 추가)
⭐ dmg-stamp.heal · 회복 도장 (#20)
밝은 녹색 + 작은 '회복' 태그. 본체 직접 피해와 같은 슬롯 — 빨강이 우선.
⭐ hp-bar-heal · HP 바 녹색 오버레이 (#20)
시작 HP → 현재 HP (회복) 구간 녹색 슬라이스. 빨강 데미지의 역방향.
⭐ dmg-stamp.curse · 저주 태그 (#23)
기존 보라 도장에 작은 '저주' 태그 추가 — 충성 도장과 동일 패턴.
⭐ elim-stamp · 보드 파괴 탈락 도장 (#24)
보드 외곽 파괴로 사라진 유닛 카드 중앙에 큰 회색 '탈락' 사선 도장.
⭐ caster-spotlight · 강화 펄스 (#14)
스킬 시전자 카드 — 황금 외곽 ring + 황금/보라/푸른 다중 글로우 펄스. brightness 1.35 + saturate 1.4.
⭐ cell-self-attackable · 제자리 공격범위 (#8)
캐릭터 딕셔너리 미리보기에서, 자기 자리가 공격범위에 포함되면 황금 ring + inset glow 로 강조.
⭐ sp-num-tick-up · 흰색 팝 (#7)
SP 숫자에 마법구가 흡수될 때 — 1.55× 스케일 + 흰색 강한 발광 (drop-shadow 14+22) + 황금 텍스트 헤일로 (#fde68a 30+48).
⭐ radial-action-menu · 보드 위 부채꼴 (#9)
유닛을 누르면 위쪽으로 펼쳐지는 라디얼 메뉴 — 이동/공격/스킬 미니 버튼이 -135°/-90°/-45° 각도로 fan.
⭐ block-bright · 팀 dim/brighten (#18)
팀모드 비활성 플레이어 카드는 opacity 0.78 / saturate 0.9 로 살짝 dim. 피격/회복 순간만 풀 밝기로 복귀 (1.8s).
