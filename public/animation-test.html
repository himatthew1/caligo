🎬 CALIGO 애니메이션 테스트

모든 시각 효과 미리보기 — 버튼을 눌러 재생

좌측 카드는 실제 게임의 컴포넌트(프로필 카드 / 보드 셀 등) 와 동일한 마크업을 사용합니다. 재생 버튼을 누르면 해당 애니메이션 클래스가 적용되었다 사라집니다.
프로필 효과 데미지 도장 HP 바 상태이상 (저주·회복) 보드 셀 풀스크린 오버레이 기타 → 사운드 테스트
① 프로필 카드 효과
profile-hit (피격)
유닛이 공격 받았을 때 — 빨간 흔들림 + 금색 플래시 (1.8s)
heal-flash (회복)
신성·약초학 회복 — 초록 플래시 (1.8s)
profile-protected (보호됨)
공격 받았으나 0 피해 — 호위무사 가로채기·아이언스킨·폭정·그림자 등. 청록 글로우 + 방패 아이콘 (1.5s)
curse-active (저주 상시)
저주 상태 캐릭터 — 진한 보라 배경 + 화염 불씨 파티클 (아래에서 위로 떠오르며 페이드아웃)
② 데미지 도장
dmg-stamp (일반 데미지)
피격 시 카드 우상단 빨간 사선 도장. 다음 턴 시작까지 유지
dmg-stamp.curse (저주 데미지)
저주 지속 데미지 — 보라 도장. 3초 후 자동 페이드 아웃
③ HP 바
hp-bar-damage (이번 턴 감소분)
이번 턴에 받은 데미지를 빨간 오버레이로 표시 — 펄스 애니메이션 1.4s
④ 상태이상 효과
heal-sparkle (회복 파티클)
회복 시 카드 위 반짝이 6개
⑤ 보드 셀 효과
cell-hit-shake (셀 흔들림)
공격 받은 셀 좌우 흔들림
marked-move-flash (표식 적 이동)
표식된 적이 이동했을 때 도착 셀 노란 플래시
teammate-move-flash (팀원 이동 도착)
팀원이 이동한 도착 셀 강조
teammate-move-trail (팀원 이동 잔상)
팀원이 떠난 자리에 보라 잔상
board-shake (보드 전체 흔들림)
보드 축소 실행 시 전체 흔들림 1초
⑥ 풀스크린 오버레이
game-start (전투 개시)
게임 시작 시 큰 텍스트 페이드 인 / 아웃 (1.8s)
shrink-intro (보드 파괴 시작)
보드 축소 카운트다운 10턴 시점 — 빨간 위협 톤 (2.2s)
stalemate (1대1 대치)
양쪽 1유닛만 남았을 때 — 5턴 후 축소 안내
⑦ 기타
shrink-warning (보드 상단 경고)
"외곽 파괴까지 N턴" — 펄스 강조 (1.2s)
pulse-end-turn (턴 종료 부드러운 강조)
행동 모두 막힌 상태에서 턴 종료 버튼이 부드러운 노란 글로우 (펄스 애니메이션 없음)
profile-hit-pulse-end-turn (위 효과 한꺼번에)
N턴 경고 + 턴 종료 강조 — 동시에 보이는 모습
