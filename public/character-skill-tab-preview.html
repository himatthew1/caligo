캐릭터 전용 스킬 탭 — v2

사용자 이미지 피드백 반영판. 헤더 (이름·티어·HP) 제거 → 공간 확보. 스킬명은 캐릭터 슬라이더의 미니헤더 클래스 그대로 사용. 시전 버튼은 우측 하단 작게. 시전 불가 시 엔트리는 딤되지만 시전 불가 버튼은 인디고로 선명, 사유는 버튼 좌측에 배치.

v1 → v2 변경 사항

  • ❌ 캐릭터 헤더 (포트레이트 / 이름 / 티어 / HP / 보유 SP) 전부 제거 → 스킬 엔트리만 표시
  • ✅ 스킬명 = 캐릭터 슬라이더의 .mini-header-{free/once/action} 클래스
  • ✅ 스킬 유형 텍스트 = 게임 통일 표기 (자유시전형 / 자유시전·1회 / 행동소비형)
  • ✅ SP 라벨 = SP 2 소모 / SP 소모 없음 (게임 통일 표기)
  • ✅ 스킬 설명 = 캐릭터 슬라이더의 정규 본문 그대로 (코드 CHAR_DETAILS 의 desc/body)
  • ✅ 시전 / 시전 불가 버튼 — 작게 + 우측 하단 배치 (여백 축소)
  • ✅ 시전 불가 시 엔트리 전체 딤, 단 시전 불가 버튼은 채도 100% 인디고 + 사유는 버튼 좌측
