게임 내부 (mark-anim.js / nightmare-anim.js / playSfxTorturerMark / playSfxNightmare) 와 동일한 모듈·사운드 사용.

① 표식 발동 — 픽셀 정렬 슬램 + 표식 길게 붉은 글로우

인두는 픽셀 기반 위치 (top:-50px right:0 width:18px height:64px) 로 cell-mark CSS (top:2px right:3px font:0.75rem) 와 정확히 매칭 — head 가 표식 위치를 픽셀 단위로 덮음.
중력 가속 (15% = 270ms) 으로 슬램 임팩트 + 셀 흔들림 + 치이익 시즐.
🎯 표식 이모지는 2.6초에 걸쳐 진하게 빛나다 천천히 복귀 — 8% 정점 → 40% 강한 글로우 유지 → 60% 옅어짐 → 100% 원상태.
이후 들어올림 거리 절반 (translateY -180%) 으로 부드럽게 사라짐.

노란 외곽선이 표식이 새겨질 대상 셀. 인두 낙하 후 ⛓ imprint 가 잔류.

② 악몽 시전 — 옵션 A 확정, 나선 오퍼시티 5단계 비교

옵션 A (2-arm 아르키메데스, 6.0s 회전) 확정. 이 안에서 나선 오퍼시티만 5단계로 비교해서 가장 적절한 존재감 선택.
유닛/🎯표식/설치물은 z-index 5~7 로 자동 위에. 표식 🎯 위치 고정 (보라 breathing 글로우만).
시연 케이스 — B1 표식+🪤덫 · D1 표식+💣폭탄 · A3 표식+🐀쥐 · E5 표식+커맨더 버프 셀 · E4 지휘관 📋.

5개 레벨 차례로 클릭해 비교 후 게임 기본값으로 확정할 레벨 알려주세요.

③ 연속 시연 — 표식 → 악몽

실제 게임 흐름: 공격으로 표식 → 다음 턴에 악몽 시전. 두 애니메이션이 자연스럽게 이어짐.
