인스턴트 SP 애니메이션 — 기존 vs 신규

팀 시점을 토글해 좌우 양 팀에 인스턴트 SP가 쌓이고 사용되는지 확인할 수 있습니다. 버튼을 눌러 두 패널의 동작을 비교해보세요.
시점 : 현재 시점 : 블루팀

기존 동작

• pip 이 SP 숫자로 비행하며 흡수됨
• 소비 순서: 숫자에서 먼 쪽부터
• 팀전 위치 버그: 항상 왼쪽(블루)에서만 재생
블루팀 SP
3
레드팀 SP
3

신규 동작

• pip 이 제자리에서 팝 (1.85배 부풀어 → 충격파 + 스파크 + dust)
• 이펙트: 중앙 플래시 + 링 2개 + 스파크 12갈래 + dust 10개
• 사운드: 인게임 _spInstantConsume(variant 3) 그대로 (1800/2400/3000Hz 3음 캐스케이드)
• 소비 순서: 숫자에 가까운 쪽부터, 한 번에 하나씩
• 잔여 pip + 숫자가 부드럽게 슬라이드 (FLIP)
• 팀전 위치 수정: 내 팀 쪽에서 재생
블루팀 SP
3
레드팀 SP
3
차이점 요약
  • 위치 버그 — 기존: 시점 무관 항상 LEFT 트레이에서 재생 / 신규: 내 팀 슬롯의 트레이에서 재생 (블루=왼쪽, 레드=오른쪽)
  • 애니메이션 — 기존: 숫자로 비행 (멀리 있는 pip 부터) / 신규: 제자리 팝 + 잔여 슬라이드 (가까운 pip 부터)
  • 리듬 — 기존 130ms, 신규 200ms 간격 (한 번에 하나씩, 명확한 호흡)
  • 잔여 SP 정리 — 신규: pip 한 개 사라질 때마다 남은 pip 들 + 숫자 가 280ms 동안 자연스럽게 모임
